# FRA-App-Trial
# Fire Risk Assessment Application

A comprehensive fire risk assessment tool with a 60-day trial period.

## Features
- Site and date tracking
- Hazard tag management
- Area-based organization
- Significant findings with risk levels
- Image capture and compression
- Auto-save functionality
- Export to PDF
- Mobile responsive design
- 60-day trial with countdown timer

## Trial Information
- Trial period: 60 days from first use
- Countdown displayed in top right corner
- Application locks after trial expires
- Contact Faeynix Artifex to continue using after trial

## Usage
Simply open the `index.html` file in a modern web browser (Chrome, Safari, Edge, Firefox).

## Browser Requirements
- Chrome 90+
- Safari 14+
- Edge 90+
- Firefox 88+

## Data Storage
All data is stored locally in your browser's localStorage. No data is sent to external servers.
